This design system uses CSS custom properties for theming. The entire UI adapts when you switch themes — no JavaScript re-rendering required. Components are built with semantic HTML and atomic CSS classes, making them portable to any framework.
:root {
--accent: #5e6ad2;
--text-primary: #f7f8f8;
--bg-base: #0f1011;
--border: rgba(255,255,255,0.08);
}

[data-theme="midnight"] {
--accent: #7850ff;
--text-primary: #eeeaff;
--bg-base: #0e0e15;
}
